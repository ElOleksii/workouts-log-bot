## Installation

1. Clone the repository

```bash
git clone https://github.com/ElOleksii/workouts-log-bot
cd workout-log-bot
```

2. Install dependencies
1. Clone the repository

```bash
npm install
```

3. Create file .env in the root and pass the bot's token

```
BOT_TOKEN=your_bot_token
```

4. Run code in dev mode

```bash
npm run dev
```
